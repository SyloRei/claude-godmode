  `userConfig.model_profile` (string, default `"balanced"`, options
  `quality|balanced|budget`). The shape is locked.
- The README and CHANGELOG copy already describes the field correctly.
  Removing the doc claims would lose a deliberate v2 design feature.
- The change is one JSON object insert; minimal blast radius.

cd /Users/sylorei/pet-projects/claude-godmode
